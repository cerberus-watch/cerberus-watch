# Athena Head

The personal safety and reconnaissance component of Cerberus, designed to empower individuals in online dating situations.

## Features

- **Flexible Input**: Accepts a name, phone number, social media link, or dating profile link as a starting point.
- **Sequential Analysis**: Performs a three-step analysis to build a comprehensive profile:
    1.  **Identity Verification**: Cross-references information to verify the individual's identity.
    2.  **Background Information**: Gathers publicly available information, leveraging the "Recon" head.
    3.  **Red Flag Analysis**: Analyzes the gathered information for potential red flags using LLM integration from the "Aegis" head.
- **Safety Score**: Provides a high-level summary and a safety score to help users make informed decisions.
