# Cerberus Assets

This directory contains various assets used throughout the Cerberus system.

## Directory Structure

- **images/**: Graphics, logos, and other image assets
- **icons/**: UI icons and symbols
- **fonts/**: Typography assets
- **css/**: Stylesheets for web interfaces
- **js/**: JavaScript libraries and utilities

## Usage Guidelines

When adding new assets, please follow these guidelines:
- Use descriptive filenames
- Include source information where applicable
- Optimize images for web use
- Document any licensing restrictions
